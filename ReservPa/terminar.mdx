# terminar
se utiliza en bucles para salir inmediatamente del bucle y continuar con la ejecución del código después del bucle.

## Ejemplo📝


```dummy
funcion ejemploTerminarEnHacerMientras() {
  imprimir("Ejemplo de terminar en un bucle hacer mientras en dummycode:");

  var i = 0;
  hacer {
    si (i === 4) {
      imprimir("Se encontró el valor 4. Terminando el bucle.");
      terminar;
    }
    imprimir("Valor de i: " + i);
    i++;
  } mientras (i < 5);

  imprimir("El bucle hacer mientras ha terminado.");
}

ejemploTerminarEnHacerMientras();
```
Salida:

```dummy

//Valor de i: 0
//Valor de i: 1
//Valor de i: 2

//El bucle hacer mientras ha terminado.
```
