# verdadero
se utiliza para representar el valor booleano, verdadero indica que una afirmación o condición es verdadera.

En dummycode tenemos dos maneras diferentes para esta palabra reservada
## Ejemplo📝

```mdx-code-block
<BrowserWindow title=''>
<Tabs>
<TabItem value="var1" label="Variante 1">
```

```dummy
global esVerdadero = verdadero; // Declaración de una variable global con el valor booleano

funcion ejemploVerdadero() {
  imprimir("Ejemplo de verdadero en dummycode:");

  si (esVerdadero === verdadero) {
    imprimir("La afirmación es verdadera.");
  } sino {
    imprimir("La afirmación no es verdadera.");
  }
}

ejemploVerdadero();
```
Salida:

```dummy
//Ejemplo de verdadero en dummycode:
//La afirmación es verdadera.
```
