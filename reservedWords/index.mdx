# Palabras Reservadas
Las palabras reservadas son palabras clave o identificadores que tienen un significado especial, están reservadas para realizar tareas específicas dentro del código. Estas palabras no pueden ser utilizadas como nombres de variables, funciones, clases u otros identificadores personalizados en el programa, ya que su uso está predeterminado por el lenguaje para llevar a cabo funciones específicas.

## Palabras reservadas para control de flujo
Estas palabras se utilizan para controlar el flujo de ejecución del programa mediante condicionales y bucles.

Ejemplos: 'si', 'osi', 'sino', 'hacer', 'mientras', 'para', 'elegir', 'caso', etc.

## Palabras reservadas para tipos de datos:
Estas palabras se utilizan para definir los tipos de datos de las variables en el programa.

Ejemplos: 'Texto', 'Lista', 'Numero', 'Logico', 'Fecha', 'Mate', 'Objeto'.

## Palabras reservadas para variables y funciones:
Estas palabras se utilizan para declarar variables, funciones y especificar su alcance y comportamiento.

Ejemplos: 'const', 'global', 'var', 'funcion', 'crear', etc.

## Palabras reservadas para valores especiales
Estas palabras representan valores especiales o estados en el programa.

Ejemplos: 'nulo', 'indefinido', 'verdadero', 'cierto', 'falso', 'NeN', etc.
