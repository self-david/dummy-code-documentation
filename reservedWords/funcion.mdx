# funcion
Una función es un bloque de código que realiza una tarea específica y puede ser invocada (llamada) en cualquier lugar del programa. 

## Ejemplo📝

```dummy
funcion ejemploFuncion() {
  imprimir("Esta es una función en dummycode");

  // Definición de una función con "function"
  funcion saludar(nombre) {
    imprimir("Hola, " + nombre + "!");
  }

  // Llamada a la función "saludar"
  saludar("Juan");
}

ejemploFunction();
```
Salida:

```dummy
// Esta es una función en dummycode
// Hola Juan !
```
