# no
Es una variante de la negacion "!" y puede ser usada para representar dicha negacion de una forma mas legible

## Ejemplo📝

```dummy
const esFalso = no verdadero

imprimir('la variable es:' + esFalso)
```
Salida:

```dummy
// la variable es: falso
```
