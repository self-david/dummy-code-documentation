# falso
falso se utiliza en expresiones lógicas y condicionales para representar el valor booleano, que indica una afirmación o condición que es falsa o no se cumple.

## Ejemplo📝

```dummy
var condicion = verdadero; // Declaración de una variable local

funcion ejemploFalso() {
  imprimir("Ejemplo de 'falso' en dummycode:");

  si (no condicion) { // tambien puedes escribirlo como '!condicion'
    imprimir("La condición es falsa.");
  } sino {
    imprimir("La condición es verdadera.");
  }
}

ejemploFalso();
```
Salida:

```dummy
// Ejemplo de 'falso' en dummycode:
// La condición es falsa.
```
