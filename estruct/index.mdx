# Estructuras de control
Las estructuras de control son bloques de código que te permiten controlar el flujo de ejecución de un programa en función de ciertas condiciones o iterar sobre un conjunto de instrucciones múltiples veces. Estas estructuras son fundamentales para la programación y ayudan a tomar decisiones y repetir tareas de manera eficiente. 

Hay tres estructuras de control principales.

## Estructuras de Control de Selección (Condicionales)
si: Permite ejecutar un bloque de código si una condición es verdadera. También puede estar acompañado de un bloque "sino" que se ejecutará si la condición es falsa.

osi: Puede ser utilizado en conjunto con "si" para comprobar múltiples condiciones en secuencia.

sino: Se utiliza con "si" para especificar un bloque de código que se ejecutará si la condición es falsa.

elegir caso: Permite tomar decisiones basadas en múltiples valores posibles de una variable.

## Estructuras de Control de Repetición (Bucles)
para: Se utiliza para ejecutar un bloque de código un número específico de veces. Generalmente, se usa con un contador.

mientras: Permite ejecutar un bloque de código mientras una condición sea verdadera.

hacer-mientras: Similar a "mientras", pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se verifica después de la ejecución.

## Estructuras de Control de Salto
terminar: Se utiliza para salir inmediatamente de un bucle, interrumpiendo su ejecución.

continuar: Se utiliza para saltar a la siguiente iteración de un bucle, omitiendo el resto del código en esa iteración particular.

regresar: Se utiliza para devolver un valor y salir de una función.
