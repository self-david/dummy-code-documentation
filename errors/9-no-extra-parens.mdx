# Parentesis innecesarios
Esto se produce cuando utilizas paréntesis que no son esenciales para la operación que estás realizando en tu código.
Los paréntesis suelen utilizarse para agrupar expresiones y establecer el orden de ejecución,
pero cuando se utilizan de forma innecesaria, pueden complicar el código sin añadir valor.

## Ejemplo📝

```dummy
var resultado = (5 + 3);
```
En este ejemplo, los paréntesis alrededor de la expresión `5 + 3` son innecesarios y provocan el error.

## Explicación
En programación, los paréntesis son útiles para aclarar la prioridad de las operaciones o para agrupar expresiones en casos específicos.
Sin embargo, cuando se usan sin motivo, pueden dificultar la lectura y comprensión del código.

## Causa
Esto suele ocurrir debido a un error tipográfico o a una confusión sobre la necesidad de los paréntesis en una expresión.

## Solución
 Para solucionar esto, simplemente elimina los paréntesis innecesarios y deja solo aquellos que son esenciales para la operación que deseas realizar.
 Aquí tienes un ejemplo corregido:
```dummy
var resultado = 5 + 3; // Hemos eliminado los paréntesis innecesarios.
```
