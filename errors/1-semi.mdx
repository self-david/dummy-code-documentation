# Punto y coma
Este error ocurre cuando olvidas colocar un punto y coma (;) al final de una línea de código donde debería estar presente según las reglas del lenguaje de programación.


## Ejemplo📝
Supongamos que estás escribiendo código en tu lenguaje de programación y tienes esto:
```dummy
const variable = 42
imprimir(variable)
```
Aquí, falta el punto y coma al final de la línea. Esto es lo que causa el error.

## Explicación
El punto y coma (;) se usa para indicar el final de una instrucción o línea de código.
Cuando falta el punto y coma, el lenguaje puede confundirse y no entender correctamente dónde termina una instrucción y comienza la siguiente.

## Causa
Este error suele ocurrir debido a un olvido simple al escribir código.

## Solución
Para solucionar el error, simplemente debes agregar el punto y coma (;) al final de la línea que falta. Aquí está el código corregido:

```dummy
const variable = 42;
imprimir(variable);
```
