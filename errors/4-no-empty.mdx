# bloque vacio
Este error se produce cuando tienes una parte de tu código que no hace nada y está en blanco. En otras palabras, no contiene instrucciones o acciones que el lenguaje de programación pueda ejecutar.

## Ejemplo📝
Imagina que tienes una función vacía como esta:
```dummy
funcion suma(a, b) {

}
```
o un bloque `si` que no hace nada
```dummy
si (edad > 10) {

}
```
En este ejemplo, la función `suma` y el bloque `si` no realizan ninguna acción, y eso causa el error

## Explicación
Escribimos código para que la computadora realice tareas específicas. Si tienes una parte de tu código que está vacía, no le estás diciendo a la computadora qué hacer, y eso puede ser un error.

## Causa
Este error suele ocurrir cuando creas una estructura de código, como una función o un bucle, pero olvidas agregar las instrucciones necesarias en su interior.

## Solución
Para solucionar el error, debes asegurarte de que la estructura de código contenga instrucciones o acciones válidas. Aquí tienes un ejemplo de cómo corregirlo:

```dummy
funcion suma() {
  regresar a + b
}
```

Tambien es posible que no tengas listo el codigo que escribiras, pero sabes que necesitas este bloque, para estos caso puedes escribir un comentario que explique que se hara.
```dummy
si (edad > 10) {
  // aqui le daremos permisos para iniciar sesion
}
```
