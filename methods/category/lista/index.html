Lista

5 minutes to learn the most important Docusaurus concepts.

📄️ posicion

Te permite acceder a un elemento específico en una Lista según su índice. La principal diferencia entre este método y la forma tradicional de acceder a un elemento es que maneja índices negativos de manera especial y permite acceder a elementos desde el final de la Lista hacia atrás.

📄️ concatenar

Se utiliza para combinar dos o más arreglos (listas) en un solo arreglo nuevo. No modifica los arreglos originales, sino que crea uno nuevo con todos los elementos de los arreglos originales concatenados en el orden en que se pasaron como argumentos.

📄️ todos

Se utiliza para verificar si todos los elementos de un arreglo (también conocido como lista) cumplen con una condición específica. Retorna verdadero si todos los elementos cumplen con la condición y falso en caso contrario. No modifica el arreglo original y toma como argumento una función que define la condición de verificación.

📄️ llenar

Modifica el arreglo original y toma uno o dos argumentos: el valor con el que deseas llenar los elementos y los índices opcionalmente que especifican en qué parte del arreglo deseas realizar el llenado.

📄️ buscar

Busca el primer elemento que cumple con la condición o error si ningún elemento cumple con la condición. No modifica el arreglo original y toma como argumento una función que define la condición de búsqueda.

📄️ buscarIndice

Retorna el índice del primer elemento que cumple con la condición o -1 si ningún elemento cumple con la condición. No modifica el arreglo original y toma como argumento una función que define la condición de búsqueda.

📄️ aplanar

Se utiliza para "aplanar" un arreglo (también conocido como lista) que contiene arreglos anidados, es decir, convertir una estructura de datos jerárquica en un arreglo plano. Este método crea un nuevo arreglo con todos los elementos de los arreglos anidados colocados en un solo nivel de profundidad. No modifica el arreglo original.

📄️ paraCada

Se utiliza para recorrer (o iterar) un arreglo (también conocido como lista) y ejecutar una función proporcionada una vez por cada elemento en el arreglo.

📄️ incluye

Se utiliza para verificar si un arreglo (también conocido como lista) contiene un elemento específico. Retorna verdadero si el elemento está presente en el arreglo y falso si no lo está.

📄️ obtenerIndice

Se utiliza para buscar la primera aparición de un elemento específico en un arreglo (también conocido como lista) y retornar el índice (la posición) de ese elemento. Si el elemento no se encuentra en el arreglo, retorna -1.

📄️ esLista

se utiliza para verificar si un valor dado es un arreglo (también conocido como lista). Retorna verdadero si el valor es un arreglo y falso si no lo es.

📄️ unir

Se utiliza para unir todos los elementos de un arreglo (también conocido como lista) en una sola cadena de texto. Puedes especificar un separador opcional que se colocará entre los elementos en la cadena resultante.

📄️ mapear

Se utiliza para crear un nuevo arreglo (también conocido como lista) a partir de otro arreglo existente, aplicando una función proporcionada a cada elemento del arreglo original. Esta función recibe un elemento como argumento y retorna un nuevo valor, y el método mapear crea un nuevo arreglo con los valores retornados por la función para cada elemento del arreglo original.

📄️ sacar

Este método modifica el arreglo original al eliminar el elemento, reduciendo la longitud del arreglo en 1.

📄️ agregar

Modifica el arreglo original al agregar los elementos y retorna la nueva longitud del arreglo después de la operación.

📄️ invertir

Se utiliza para invertir el orden de los elementos en un arreglo (también conocido como lista). Este método modifica el arreglo original al cambiar el orden de los elementos.

📄️ sacarPrimero

Este método modifica el arreglo original al eliminar el primer elemento, reduciendo la longitud del arreglo en 1.

📄️ cortar

se utiliza para crear una copia superficial de una porción de un arreglo (también conocido como lista) y retornarla como un nuevo arreglo.

📄️ alguno

Se utiliza para verificar si al menos un elemento de un arreglo (también conocido como lista) cumple con cierta condición proporcionada a través de una función de prueba.

📄️ ordenar

Se utiliza para ordenar los elementos de un arreglo (también conocido como lista) de manera predeterminada o aplicando una función de comparación personalizada.

📄️ empalme

Se utiliza para modificar un arreglo (también conocido como lista) al agregar, eliminar o reemplazar elementos en una posición específica del arreglo. Este método puede tomar varios argumentos y permite una amplia variedad de operaciones en el arreglo original.

📄️ aTexto

Se utiliza para convertir un arreglo (también conocido como lista) en una cadena de texto. Este método retorna una cadena que contiene los elementos del arreglo separados por comas (,).

📄️ agregarPrimero

Se utiliza para agregar uno o más elementos al principio de un arreglo (también conocido como lista).
