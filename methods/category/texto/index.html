Texto

5 minutes to learn the most important Docusaurus concepts.

📄️ Posicion

Este método toma un valor entero y devuelve una nueva cadena que consta de la única unidad de código UTF-16 ubicada en el desplazamiento especificado. Este método permite números enteros positivos y negativos. Los enteros negativos cuentan hacia atrás desde el último carácter de cadena.

📄️ Concatenar

Se utiliza para combinar o unir dos o más cadenas de texto en una sola cadena. Imagina que tienes dos o más cadenas de texto y deseas unirlas para formar una cadena más larga.

📄️ TerminaCon

El método determina si una cadena termina con los caracteres de una cadena específica, devolviendo verdadero o falso según corresponda.

📄️ Incluye

Puedes usarlo para verificar si una cadena de texto contiene otra cadena específica. Si la cadena está presente, el método retorna verdadero; de lo contrario, retorna falso.

📄️ IndiceDe

Puedes usarlo para encontrar la posición de la primera aparición de una subcadena en una cadena de texto. Si la subcadena está presente, el método retorna el índice de la primera ocurrencia; de lo contrario, retorna -1.

📄️ UltimoIndiceDe

Dicho método funciona de manera similar. Puedes usarlo para encontrar la posición de la última aparición de una subcadena en una cadena de texto. Si la subcadena está presente, el método retorna el índice de la última ocurrencia; de lo contrario, retorna -1.

📄️ Coincidir

Puedes usarlo para buscar todas las coincidencias de un patrón específico en una cadena de texto y obtener un arreglo con todas las coincidencias encontradas.

📄️ CoincidirTodo

Puedes utilizarlo para buscar todas las coincidencias de un patrón específico en una cadena de texto y obtener un iterador que te permita acceder a todas las coincidencias encontradas.

📄️ RellenarFin

Dicho metodo te permite añadir caracteres al final de una cadena de texto hasta que alcance una longitud determinada. Debes especificar la longitud deseada y el carácter que deseas agregar.

📄️ RellenarInicio

Este permite agregar caracteres al inicio de una cadena de texto hasta que alcance una longitud determinada. Debes especificar la longitud deseada y el carácter que deseas agregar.

📄️ Repetir

Este método te permite crear una nueva cadena de texto al repetir una cadena existente múltiples veces. Imagina que tienes una palabra y quieres duplicarla varias veces para formar una nueva cadena más larga.

📄️ Reemplazar

Este se utiliza para buscar y reemplazar una parte específica de una cadena de texto por otra cadena o valor. Imagina que tienes una frase y quieres cambiar una palabra en particular por otra.

📄️ reemplazarTodo

Este toma dos argumentos: el primer argumento es la subcadena que deseas reemplazar, y el segundo argumento es la subcadena con la que deseas reemplazar todas las ocurrencias de la primera subcadena. Este método busca todas las coincidencias de la primera subcadena en la cadena original y las reemplaza con la segunda subcadena.

📄️ Buscar

Se utiliza para buscar un patrón en una cadena de texto y devuelve la posición de la primera coincidencia.

📄️ Cortar

Se utiliza para extraer una parte específica de una cadena de texto sin modificar la cadena original. Imagina que tienes una frase y quieres obtener una parte específica de esa frase.

📄️ Dividir

Se utiliza para dividir una cadena de texto en un arreglo de subcadenas, utilizando un separador especificado. Imagina que tienes una frase y quieres dividirla en palabras individuales.

📄️ IniciaCon

Se utiliza para verificar si una cadena de texto comienza con un valor específico. Imagina que tienes una palabra y quieres saber si empieza con una letra o un conjunto de letras en particular.

📄️ Subcadena

Se utiliza para extraer una parte específica de una cadena de texto. Imagina que tienes una frase y quieres obtener una porción determinada de esa frase.

📄️ aMinusculas

Se utiliza para convertir una cadena de texto a minúsculas. Imagina que tienes una palabra o una frase en mayúsculas y quieres convertirla a minúsculas.

📄️ aMayusculas

Se utiliza para convertir una cadena de texto a mayúsculas. Imagina que tienes una palabra o una frase en minúsculas y quieres convertirla a mayúsculas.

📄️ aTexto

Se utiliza para convertir cualquier cosa en una cadena de texto. I

📄️ Limpiar

Se utiliza para eliminar los espacios en blanco al inicio y al final de una cadena de texto. Imagina que tienes una cadena de texto con espacios en blanco extra y quieres eliminarlos.

📄️ LimpiarFinal

Se utiliza para eliminar los espacios en blanco al final de una cadena de texto. Imagina que tienes una cadena de texto con espacios en blanco al final y deseas eliminarlos.

📄️ LimpiarInicio

Se utiliza para eliminar los espacios en blanco al inicio de una cadena de texto. Imagina que tienes una cadena de texto con espacios en blanco al inicio y deseas eliminarlos.
