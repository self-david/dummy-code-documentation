Fecha

📄️ obtenerFecha

Te permite obtener el día del mes representado por la fecha almacenada en el objeto Fecha.

📄️ dia

Se utiliza para obtener el día del mes de una fecha específica. Retorna un número entero que representa el día del mes (1-31) de la fecha proporcionada.

📄️ añoCompleto

Sirve para obtener el año de una instancia de un objeto Fecha. Devuelve un número de cuatro dígitos que representa el año correspondiente.

📄️ hora

Se usa para obtener la hora de una instancia de un objeto Fecha. Devuelve el valor de la hora en el formato de 24 horas (de 0 a 23).

📄️ miliseg

Con este se obtiene los milisegundos de una instancia de un objeto Fecha. Devuelve un número entero que representa los milisegundos en la fecha y hora específica.

📄️ minutos

Si para obtener los minutos de una instancia de un objeto Fecha. Devuelve un número entero que representa los minutos dentro de la hora de la fecha.

📄️ mes

Se usa para obtener el mes de una instancia de un objeto Fecha. Devuelve un número entero que representa el mes, donde enero es 0 y diciembre es 11.

📄️ segundos

Se utiliza para obtener los segundos de una instancia de un objeto Fecha. Devuelve un número entero que representa los segundos en la hora actual.

📄️ tiempo

Se utiliza para obtener el valor numérico que representa el tiempo transcurrido en milisegundos. Este valor es comúnmente utilizado para comparar y calcular diferencias de tiempo.

📄️ Ahora

Se utiliza para obtener el valor numérico correspondiente a la fecha y hora actual en milisegundos. A diferencia del método tiempo, no requiere crear una instancia de Fecha y se llama directamente en el objeto Fecha.

📄️ textoAFecha

Toma una cadena de fecha en un formato específico y la convierte en un objeto Fecha que representa esa fecha y hora.

📄️ actualizarFecha

Establece el día del mes de un objeto Fecha. Permite modificar el día del mes de una fecha existente, sin afectar a otros componentes de la fecha como el mes o el año.

📄️ actualizarAño

Se utiliza para establecer el año en un objeto Date. Puedes usar este método para cambiar el año de una fecha existente o para crear una nueva fecha con el año deseado.

📄️ actualizarHora

Con este se pueden establecer las horas en un objeto Fecha. Puedes usar este método para cambiar las horas de una fecha existente o para crear una nueva fecha con las horas deseadas.

📄️ actualizarMiliseg

Puedes usar este método para cambiar los milisegundos de una fecha existente o para crear una nueva fecha con los milisegundos deseados.

📄️ actualizarMinutos

Puedes usar este método para cambiar los minutos de una fecha existente o para crear una nueva fecha con los minutos deseados.

📄️ actualizarMes

Puedes usar este método para cambiar el mes de una fecha existente o para crear una nueva fecha con el mes deseado.

📄️ actualizarSegundos

Puedes usar este método para cambiar los segundos de una fecha existente o para crear una nueva fecha con los segundos deseados.

📄️ actualizarTiempo

Se utiliza para establecer una fecha y hora específicas a partir de un valor de tiempo en milisegundos. Este método toma un único argumento que representa el valor de tiempo en milisegundos.

📄️ fechaATexto

Se utiliza para obtener una representación de cadena de la parte de la fecha de un objeto Fecha. Esta representación de cadena incluye el día de la semana, el mes, el día del mes y el año en formato legible.

📄️ aTexto

se utiliza para obtener una representación de cadena de texto de un objeto Fecha. Esta representación de cadena incluye tanto la parte de la fecha como la parte de la hora del objeto Fecha en un formato específico.
