# Logico
Se utiliza para representar valores de verdad, es decir, afirmaciones que pueden ser verdaderas o falsas.

Aquí tienes algunas características clave del tipo de dato "Lógico":

## Valores: 
Un "Lógico" puede tener dos valores posibles: verdadero  o falso. Estos valores representan afirmaciones verdaderas o falsas, respectivamente.

## Operadores Lógicos: 
Se utilizan operadores lógicos como y (&&), o (||) y no (!) para realizar operaciones lógicas en valores booleanos. Estos operadores permiten combinar y evaluar afirmaciones lógicas de manera efectiva.

## Comparaciones: 
Los valores booleanos se generan comúnmente al realizar comparaciones en el código. Por ejemplo, una comparación como 5 > 3 devuelve verdadero porque la afirmación "5 es mayor que 3" es verdadera.

## Decisiones Condicionales: 
Los valores booleanos se utilizan en declaraciones si para tomar decisiones condicionales. Por ejemplo, un bloque de código se ejecutará si una condición es verdadera, de lo contrario, se ejecutará otro bloque si es falsa.

## Variables Booleanas: 
Es común utilizar variables booleanas para mantener el estado de una aplicación o indicar si una condición se cumple o no.

## Expresiones Booleanas: 
Las expresiones booleanas se utilizan en muchas partes del código para controlar el flujo y el comportamiento de un programa.

## Ejemplo📝

```dummy
var esMayorDeEdad = verdadero; // Variable Logica

si (esMayorDeEdad) {
  imprimir("Eres mayor de edad.");
} sino {
   imprimir("Eres menor de edad.");
}
```
Salida:

```dummy
// verdadero
```
