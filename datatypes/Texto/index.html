Texto

El tipo de dato "Texto" (String) se utiliza para almacenar y manipular secuencias de caracteres. Puedes pensar en un "Texto" como una cadena de letras, números, espacios y caracteres especiales.

Aquí tienes algunas características clave del tipo de dato "Texto":

Secuencia de Caracteres:​

Un "Texto" es una secuencia de caracteres que puede contener letras, números, espacios en blanco, signos de puntuación y otros caracteres.

Inmutabilidad:​

Los "Textos" son inmutables, lo que significa que una vez que se crea un "Texto", no se puede modificar directamente. En cambio, se crean nuevas cadenas cuando se realizan operaciones de manipulación.

Índices y Longitud:​

Puedes acceder a caracteres individuales dentro de un "Texto" utilizando índices. La longitud de un "Texto" se refiere al número de caracteres que contiene.

Concatenación:​

Puedes concatenar (unir) dos o más "Textos" para crear uno nuevo utilizando el operador + u otras funciones específicas del lenguaje.

Operaciones de Subcadena:​

Puedes extraer subcadenas de un "Texto" utilizando métodos como substring, slice o substr. Esto te permite trabajar con partes específicas de una cadena.

Comparaciones:​

Puedes comparar "Textos" utilizando operadores de comparación, como ==, ===, <, >, <=, >=, para evaluar igualdad o orden.

Búsqueda y Reemplazo:​

Puedes buscar subcadenas dentro de un "Texto" y reemplazarlas por otras utilizando funciones como indiceDe, remplazar, buscar, coincidir, entre otras.

Formato y Manipulación:​

Los "Textos" se utilizan para mostrar información formateada en interfaces de usuario y para manipular datos de texto, como nombres, direcciones, números de teléfono, etc.
