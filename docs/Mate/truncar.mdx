# truncar
Se utiliza para eliminar los decimales de un número al siguiente entero menor o igual.

## Ejemplo📝

```dummy
var numero = 5.8;
var resultado = Mate.truncar(numero);

imprimir("El redondeo hacia abajo de " + numero + " es: " + resultado);
```

Salida:

```dummy
// El redondeo hacia abajo de 5.8 es: 5
```
