# raiz
Calcula la raíz cuadrada de un número dado.

## Ejemplo📝

```dummy
var numero = 25;

var raizCuadrada = Mate.raiz(numero);

imprimir("La raíz cuadrada de " + numero + " es: " + raizCuadrada);
```

Salida:

```dummy
// La raíz cuadrada de 25 es: 5
```
