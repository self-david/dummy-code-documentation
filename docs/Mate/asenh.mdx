# asenh
Sirve para calcular el arcoseno hiperbólico de un número dado.

## Ejemplo📝

```dummy
var numero = 2.5;
var resultado = Mate.asenh(numero);

imprimir("El arcoseno hiperbólico de " + numero + " es: " + resultado);
```

Salida:

```dummy
//El arcoseno hiperbólico de 2.5 es: 1.6472311463710958
```
Recuerda que el valor del arcoseno hiperbólico se devuelve en radianes.
