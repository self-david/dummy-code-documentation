# redondeo
El método se utiliza para redondear un número al siguiente entero mayor o igual.

## Ejemplo📝

```dummy
var numero = 5.3;
var resultado = Mate.redondeo(numero);

imprimir("El redondeo hacia arriba de " + numero + " es: " + resultado);
```

Salida:

```dummy
// El redondeo hacia arriba de 5.3 es: 6
```
