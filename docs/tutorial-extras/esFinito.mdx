# EsFinito

Determina si un valor es un número finito o no. Retorna true si el valor es un número finito y false si el valor es infinito, -infinito, NeN o cualquier otro valor no numérico.

## Ejemplo📝

```mdx-code-block
<BrowserWindow title='esFinito'>
<Tabs>
<TabItem value="var1" label="Variante 1">
```

```dummy
var numero1 = 5;
var numero2 = Infinito;
var numero3 = "Hola";

imprimir("¿Es el número 5 finito? " + esFinite(numero1));
imprimir("¿Es el valor Infinity finito? " + esFinite(numero2));
imprimir("¿Es la cadena 'Hola' finita? " + esFinite(numero3));

```
Salida:

```dummy
//¿Es el número 5 finito? true
//¿Es el valor Infinity finito? false
//¿Es la cadena 'Hola' finita? false
```
