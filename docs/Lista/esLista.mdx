# esLista
se utiliza para verificar si un valor dado es un arreglo (tambi√©n conocido como lista). Retorna verdadero si el valor es un arreglo y falso si no lo es.



```dummy
// Verificar si un valor es un arreglo
var arreglo = [1, 2, 3];
var noArreglo = 'Hola, soy una cadena';

var esArreglo = lista.esLista(arreglo);
var noEsArreglo = lista.esLista(noArreglo);
imprimir(esArreglo);
imprimir(noEsArreglo)
```
Salida:

```dummy
// verdadero
// falso
```
