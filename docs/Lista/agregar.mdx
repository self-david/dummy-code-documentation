# agregar
Modifica el arreglo original al agregar los elementos y retorna la nueva longitud del arreglo después de la operación.

## Ejemplo📝

```mdx-code-block
<BrowserWindow title=''>
<Tabs>
<TabItem value="var1" label="Variante 1">
```

```dummy
// Crear un arreglo de ejemplo
var frutas = ['manzana', 'pera', 'naranja'];

// Agregar una nueva fruta al arreglo usando agregar
var nuevaFruta = 'uva';
var nuevaLongitud = frutas.agregar(nuevaFruta);

imprimir(nuevaLongitud);
```
Salida:

```dummy
// ['manzana', 'pera', 'naranja', 'uva']
```

```dummy
// Crear un arreglo de ejemplo
var frutas = ['manzana', 'pera', 'naranja'];

// Agregar múltiples frutas al arreglo usando agregar
var nuevasFrutas = ['mango', 'sandía'];
var muchasFrutas = frutas.agregar(nuevasFrutas);

imprimir(muchasFrutas);
```
Salida:
```dummy
// ['manzana', 'pera', 'naranja', 'uva', 'mango', 'sandía']
```
